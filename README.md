# freenove-ng
Curated Freenove codebase, cleaning up code, and creating both C and C++ selections

## code/
Collections exist for several languages:

  * `C`: commented and polished C code from the original Freenove repository
  * `C++`: Object-Oriented C++ code (ported from C)
  * `bash`: bash scripts using wiringPi `gpio` tool plus any supporting tools
  * `python`: commented and polished python3 scripts using GPIOZero from the original Freenove repository
